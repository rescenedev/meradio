개인정보처리방침

최종 업데이트: 2026년 6월 21일

meradio는 한국 라디오 방송을 듣는 네이티브 macOS 메뉴 바 앱입니다. meradio는 사용자의 개인정보를 필요로 하지 않도록 설계되었습니다.

수집하는 정보

없습니다. meradio는 개인정보를 수집·전송·판매·공유하지 않습니다. 계정, 분석(애널리틱스), 추적, 광고가 일절 없습니다.

기기에 저장되는 데이터

즐겨찾기, 최근 재생 목록, 볼륨, 시작 모드, 예약 설정 등 환경설정은 macOS 표준 설정 시스템(UserDefaults)을 통해 사용자의 Mac에만 로컬로 저장됩니다. 이 데이터는 기기를 벗어나지 않으며, 앱과 설정을 삭제하면 함께 제거됩니다.

네트워크 사용

meradio는 라디오를 재생하기 위해 각 방송국의 스트리밍 서버에 직접 연결합니다. 이때 발생하는 통신은 오디오 스트림을 받아오기 위한 것이며, meradio가 운영하는 서버는 없습니다. 방송국 스트림 서버에는 일반적인 인터넷 연결과 동일하게 사용자의 IP 주소 등 표준 네트워크 정보가 전달될 수 있으며, 해당 정보의 처리는 각 방송사/스트리밍 제공자의 정책을 따릅니다.

제3자 서비스

meradio에는 제3자 분석 도구, 추적기, 광고 SDK가 포함되어 있지 않습니다. 방송국 목록 및 스트림 주소는 공개된 정보(radio-korea.com)를 기반으로 앱에 내장되어 있습니다.

방침 변경

본 방침이 변경되는 경우, 변경된 내용과 새로운 "최종 업데이트" 날짜를 이 페이지에 게시합니다.

문의

개인정보나 앱에 대한 문의는 tellme@duck.com으로 보내주세요.

English summary: meradio collects no personal data, has no accounts, analytics, tracking, or ads. Preferences are stored locally via UserDefaults. The app connects directly to each station's streaming server to play audio; there is no meradio-operated server. Station data is bundled from public sources (radio-korea.com).
